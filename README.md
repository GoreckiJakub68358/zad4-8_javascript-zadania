## 📂 Struktura repozytorium

```text
/
├── zad4/      # Katalog dla zadania 4
├── zad5/      # Katalog dla zadania 5
├── zad6/      # Katalog dla zadania 6
├── zad7/      # Katalog dla zadania 7
├── zad8/      # Katalog dla zadania 8
├── index.html # Struktura strony z linkami do zadań
├── style.css  # Arkusz stylów do strony z linkami do zadań
└── README.md  # Dokumentacja projektu
```

Każdy folder zawiera:
- własny `index.html`
- pliki CSS
- pliki JavaScript (jeśli wymagane)
- wszystkie zasoby (np. zdjęcia)

Przykładowe adresy:
- Zadanie 3:  
  `.../zad3/index.html`
- Zadanie 4:  
  `.../zad4/index.html`
